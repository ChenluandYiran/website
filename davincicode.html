Davinci Code Player

• Davinci Code robot, a 7 DOF real-time board game player you might only see in the movies, is a collaborated work between Chenlu Wu and Yiran Zuo.

• Supported by vision system based on OpenCV, it can see the surrounding environment to find out the target panels, moving or flipping the target and analyzing the obtained data to make further inference.

• Two control modes: interactive sound mode and GUI mode.

• Strong interactive ability: our robot could talk to you and respond you in real-time, just like you are playing with a real friend. It’s a best companion for kids to keep them engaged and entertained, even fostered their inferential capability.

• If the player wants to keep quiet while playing, our nice friendly user interface is another great option.
